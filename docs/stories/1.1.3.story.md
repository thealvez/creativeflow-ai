**Epic:** EPIC 1 — Foundation

## 🎯 Story Summary

Install all npm dependencies for monorepo workspaces, configure shared development dependencies (TypeScript, ESLint, Prettier), and ensure all packages can build successfully.

---

## 👤 User Story

**As a** developer
**I want** all dependencies installed and configured correctly
**So that** I can start developing without dependency issues.

---

## ✅ Acceptance Criteria

### AC-1: Dependencies Installed
```
✓ npm ci succeeds without errors
✓ All workspace dependencies installed:
  - frontend: next, react, react-dom, tailwindcss
  - backend: fastify, pg, redis
  - agents: @anthropic-ai/sdk, langchain
✓ No duplicate dependencies across workspaces
✓ Lock file (package-lock.json) committed to git
```

### AC-2: Shared Dev Dependencies
```
✓ typescript 5.3+ installed (root)
✓ eslint 8.54+ installed (root)
✓ prettier 3.2+ installed (root)
✓ All packages can reference shared eslint config
✓ npm run lint works across all packages
```

### AC-3: Package.json Scripts Verified
```
✓ npm run dev starts all dev servers
✓ npm run build builds all packages
✓ npm run test runs all tests
✓ npm run lint lints all packages
✓ npm run typecheck type-checks all packages
✓ npm run format formats all code files
```

### AC-4: No Build Errors
```
✓ npm run lint passes (0 errors)
✓ npm run typecheck passes (0 errors)
✓ No security vulnerabilities (npm audit)
✓ No deprecation warnings (only advisory)
✓ All dependencies at latest stable versions
```

### AC-5: Development Environment Ready
```
✓ Node.js version verified (v20+)
✓ npm version verified (11.6+)
✓ TypeScript can compile to JavaScript
✓ ESLint can parse all files
✓ Prettier can format all files
```

### AC-6: Documentation Updated
```
✓ .npmrc created (if needed)
✓ DEVELOPMENT.md explains dependency management
✓ Package.json comments explain key dependencies
✓ .gitignore includes node_modules + lock files
```

## 🛠️ Technical Details

### **Key Dependencies**

**Frontend:**
- next 15.0.0+ (React framework)
- react 19.0.0+ (UI library)
- tailwindcss 3.4.0+ (styling)
- typescript 5.3.0+ (type checking)
- eslint 8.54.0+ (linting)

**Backend:**
- fastify 4.25.0+ (HTTP server)
- pg 8.11.0+ (PostgreSQL driver)
- redis 4.6.0+ (Redis client)
- knex 3.0.0+ (Query builder)
- bull 4.12.0+ (Job queue)

**Agents:**
- @anthropic-ai/sdk 0.25.0+ (Claude API)
- langchain 0.1.0+ (Agent orchestration)

### **Version Strategy**
- Use exact versions (^X.Y.Z) for stability
- Update packages monthly (not weekly)
- Test major version upgrades before committing
- Keep dev dependencies latest (less critical)
